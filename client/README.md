# tron-cold-sign-client

Vite + React UI. Shared crypto lives in the repo root package **`tron-cold-sign`**. From the repository root, run **`npm run build:lib`** to produce **`dist/tron-lib-esm.mjs`** before the client if you invoke `vite` without the root scripts (`npm run dev:client` / `build:client` / `test:client` run `build:lib` for you).
